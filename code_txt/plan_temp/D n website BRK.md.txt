# 4️⃣ Security Audit



Hệ thống LMS thường bị lỗi:


### 1️⃣ Bypass lesson



Ví dụ:


```
/lesson/123
```



Nếu không check enrollment:


user có thể **xem bài miễn phí**.


Middleware nên kiểm tra:


```
user enrolled?
```



---



### 2️⃣ Admin route protection



Route:


```
/admin
```



Phải kiểm tra:


```
role = ADMIN
```



---



### 3️⃣ Payment spoofing



Nếu code kiểm tra:


```
amount only
```



thì dễ bị fake.


Cần kiểm tra:


```
amount

bank reference

payment content
```
